# Developing
## Prerequisits
- [Deno](https://deno.land/)
- [npm](https://www.npmjs.com/)

## Installing
```zsh
npm install
```

## Runing frontend
```zsh
npm run dev

# or start the server and open the app in a new browser tab
npm run dev -- --open
```

## Runing backend
```zsh
npm run server
```

## Building
```zsh
npm run build
```
